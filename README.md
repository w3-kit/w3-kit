# w3-kit

CLI toolkit for web3 development.

## Install

```bash
npx w3-kit
```

## Commands

### `w3-kit init`
Scaffold a new web3 project from starter templates.

### `w3-kit add <recipe>`
Add a recipe or contract to your project.

### `w3-kit registry`
Query chain and token data from the terminal.

## Part of the w3-kit ecosystem

- [@w3-kit/ui](https://github.com/w3-kit/ui) — Component library
- [@w3-kit/registry](https://github.com/w3-kit/registry) — Chain/token data
- [w3-kit/learn](https://github.com/w3-kit/learn) — Recipes and guides
- [w3-kit/contracts](https://github.com/w3-kit/contracts) — Smart contracts
- [w3-kit/website](https://github.com/w3-kit/website) — w3-kit.com
